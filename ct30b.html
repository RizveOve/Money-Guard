All-in-one counter terminal CT30B

An integrated paperless solution for financial service counters integrated with electronic signature pads, EPPs, a touch screen, embedded keyboard, fingerprint reader that are al equipped with data encrypted storage with optimal services ratings.
The perfect combination of functionality and using a small footprint makes it deal for counter services to become clutter free and user friendly for your banking customers.

Technical Specification:

-> Controller: Android/Linux OS, Octa-core CPU 2.0GHz, 2GB memory, 16G storage;
-> Touch screen: 10.1-inch IPS, 16:9, 1280*800 pixels
-> Camera (opt.): Monocular camera (opt.)
  Binocular camera (opt.)
-> Fingerprint device: Ministry of Public Security standard
-> EPP: 12 physical keys, number keys support Braille, national encryption standard
-> Communication: Standard: RS232, USB
 Optional: WIFI, bluetooth, Rj45
-> Electronic signature: Passive electromagnetic pen
-> Sound input: Pickup, MIC phone
-> External interface: USB
-> Dimension: 355mm x 168mm x 17mm (L*W*H)



Features:

The all-in-one counter terminal realizes paperless branch, it's a low-carbon, environmentally friendly, safe, and convenient “new” financial services.


-> Integrated compact design to enhance customer experience

-> Biometric identification integration to streamline authentication procedures -> High end encryption for premium security

-> Multi-functional use with electronic signatures, EPP key pad for passwords, service evaluation option, and multimedia advertising

-> Supports remote authorization, financial IC cards and other business scenarios

-> Includes value-added services like electromagnetic dual-mode screen, biometric scanning (Face or fingerprint)

-> Unified management platform that supports remote management, batch distribution for upgrades and advertising, configuration management, and much more
